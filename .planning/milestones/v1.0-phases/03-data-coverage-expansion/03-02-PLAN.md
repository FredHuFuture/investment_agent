    BUY = "BUY"
    HOLD = "HOLD"
    SELL = "SELL"
```

Aggregation rule for FinBERT (we decide this):
- Aggregate N headline scores into a single signal:
  - mean_score = mean([+1 * score if label=="positive" else -1 * score if label=="negative" else 0 for ...])
  - If mean_score > 0.25: BUY, confidence = min(90, 50 + mean_score * 100)
  - If mean_score < -0.25: SELL, confidence = min(90, 50 + abs(mean_score) * 100)
  - Else: HOLD, confidence = 40
- Headline count >= 3 required for non-HOLD (too few headlines = unreliable local inference)

    #!/usr/bin/env python
    """Pre-download the FinBERT model so the first SentimentAgent.analyze call is fast.

    Run this once after `pip install -e .[llm-local]`:
        python scripts/fetch_finbert.py

    The model (~400 MB) will be cached in the default HuggingFace cache dir
    (typically ~/.cache/huggingface/hub/models--ProsusAI--finbert on macOS/Linux
    or %USERPROFILE%\\.cache\\huggingface\\hub\\... on Windows).
    """
    from __future__ import annotations
    import sys

    def main() -> int:
        try:
            from transformers import pipeline
        except ImportError:
            print(
                "transformers is not installed. Run:\n"
                "    pip install -e .[llm-local]\n"
                "to enable local FinBERT sentiment.",
                file=sys.stderr,
            )
            return 1

        print("Downloading ProsusAI/finbert (~400 MB)...", flush=True)
        try:
            _ = pipeline("sentiment-analysis", model="ProsusAI/finbert")
        except Exception as exc:
            print(f"FinBERT download failed: {exc}", file=sys.stderr)
            return 1

        print("FinBERT cached. SentimentAgent can now use local sentiment when ANTHROPIC_API_KEY is unset.")
        return 0

    if __name__ == "__main__":
        sys.exit(main())
    ```

    ```python
    def __init__(self, provider, news_provider=None) -> None:
        super().__init__(provider)
        self._news_provider = news_provider
        self._finbert_pipeline = None  # lazy-loaded on first use

    def _try_load_finbert(self) -> bool:
        """Return True if FinBERT is importable and pipeline loadable. Cache result."""
        global _FINBERT_IMPORT_ATTEMPTED, _FINBERT_AVAILABLE
        if _FINBERT_IMPORT_ATTEMPTED:
            return _FINBERT_AVAILABLE
        _FINBERT_IMPORT_ATTEMPTED = True
        try:
            from transformers import pipeline  # noqa: F401
            _FINBERT_AVAILABLE = True
        except ImportError:
            _FINBERT_AVAILABLE = False
            logger.info(
                "transformers not installed; FinBERT sentiment fallback disabled. "
                "Install with: pip install -e .[llm-local]"
            )
        return _FINBERT_AVAILABLE

    def _get_finbert_pipeline(self):
        if self._finbert_pipeline is None:
            from transformers import pipeline  # actually import now
            logger.info("Loading FinBERT pipeline (first call; may download ~400 MB)...")
            self._finbert_pipeline = pipeline("sentiment-analysis", model="ProsusAI/finbert")
        return self._finbert_pipeline

    async def _analyze_with_finbert(self, ticker: str, headlines: list) -> AgentOutput:
        """Run local FinBERT inference over headlines. Requires >=3 headlines for non-HOLD."""
        import asyncio as _asyncio
        MIN_HEADLINES = 3
        if len(headlines) < MIN_HEADLINES:
            return self._fallback_output(
                ticker,
                confidence=40.0,
                warning=f"FinBERT requires >= {MIN_HEADLINES} headlines; got {len(headlines)}.",
            )

        # Run inference in a thread (transformers pipelines are sync + CPU-bound)
        pipe = self._get_finbert_pipeline()
        texts = [
            (f"{h.title}. {h.snippet}" if h.snippet else h.title)[:512]  # FinBERT truncates at 512
            for h in headlines
        ]

        def _infer():
            return pipe(texts)

        results = await _asyncio.to_thread(_infer)

        # Aggregate: signed score per label, mean across headlines
        signed = []
        for r in results:
            label = r.get("label", "neutral").lower()
            score = float(r.get("score", 0.0))
            if label == "positive":
                signed.append(+score)
            elif label == "negative":
                signed.append(-score)
            # neutral contributes 0
        if not signed:
            return self._fallback_output(
                ticker, confidence=40.0, warning="FinBERT returned no parseable results.",
            )
        mean_score = sum(signed) / len(results)

        BUY_THRESHOLD = 0.25
        SELL_THRESHOLD = -0.25
        if mean_score >= BUY_THRESHOLD:
            signal = Signal.BUY
        elif mean_score <= SELL_THRESHOLD:
            signal = Signal.SELL
        else:
            signal = Signal.HOLD
        confidence = max(30.0, min(90.0, 50.0 + abs(mean_score) * 100.0))

        return AgentOutput(
            agent_name=self.name,
            ticker=ticker,
            signal=signal,
            confidence=confidence,
            reasoning=(
                f"FinBERT local inference on {len(headlines)} headlines; "
                f"mean sentiment {mean_score:+.2f} -> {signal.value}. "
                f"Model: ProsusAI/finbert."
            ),
            metrics={
                "sentiment_score": mean_score,
                "catalyst_count": 0,  # FinBERT doesn't extract named catalysts
                "headline_count": len(headlines),
                "cost_usd": 0.0,
                "model": "ProsusAI/finbert",
                "inference": "local",
            },
            warnings=["Using FinBERT local inference (ANTHROPIC_API_KEY not set)."],
        )
    ```

## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| HuggingFace Hub <-> FinBERT pipeline | Model download fetches weights from HF CDN over HTTPS on first use |
| transformers library <-> user news headlines | Untrusted text passed to local inference; transformers pipeline is safe but must be defensive |
| `[llm-local]` extra <-> default install | User must opt-in; ensures default `pip install .` stays small (~50 MB) |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-03-02-01 | Tampering | FinBERT model weights | accept | Model is Apache 2.0 from ProsusAI org, verified via HF hub checksums by transformers library automatically. We don't pin a revision — accepted risk for free-tier model. |
| T-03-02-02 | Denial of Service | First-run 400 MB download | mitigate | Pre-download script `scripts/fetch_finbert.py` lets user fetch explicitly. Log message on lazy first-load warns "may download ~400 MB". If download fails, falls back to HOLD with explicit warning — analysis pipeline keeps working. |
| T-03-02-03 | Information Disclosure | Headline text sent to transformers | accept | Inference is LOCAL — headlines never leave the machine. This is an IMPROVEMENT over Anthropic branch which sends text to Claude API. |
| T-03-02-04 | Information Disclosure | HF model download URL leaks presence | accept | Standard HuggingFace usage. Solo-operator local-first constraint means no network egress monitoring risk. |
| T-03-02-05 | Denial of Service | FinBERT inference blocks event loop | mitigate | `asyncio.to_thread(_infer)` wraps sync pipeline call; runs in thread pool. 10 headlines inference is ~200ms on CPU. |
| T-03-02-06 | Elevation of Privilege | transformers library loads arbitrary code | accept | transformers does NOT execute untrusted Python from the model — tokenizer+inference only. Safe for sentiment use case. |
